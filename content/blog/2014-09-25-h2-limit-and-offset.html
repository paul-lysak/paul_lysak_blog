--- layout: post title: H2 LIMIT and OFFSET date: '2014-09-25T15:24:00.000+03:00' author: Paul Lysak tags: - SQL - H2 modified_time: '2014-12-29T16:02:04.716+02:00' blogger_id: tag:blogger.com,1999:blog-5849718801312198988.post-6633864186063369869 blogger_orig_url: http://paullysak.blogspot.com/2014/09/h2-limit-and-offset.html --- Funny thing - H2 in PostgreSQL compatibility mode requires that OFFSET should go strictly after LIMIT (like LIMIT 25 OFFSET 50). Otherwise it can't parse the query: 
error in SQL statement "SELECT ..........   OFFSET 0  LIMIT[*] 25 "; 
expected "[, ::, *, /, %, +, -, ||, ~, !~, NOT, 
LIKE, REGEXP, IS, IN, BETWEEN, AND, OR, ROW, ROWS"
